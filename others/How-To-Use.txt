
说明 如何部署, 如何使用 MyOIDC


[myoidc-server]

1. 在MySQL中创建数据库: myoidc_server
  create database if not exists myoidc_server default character set utf8;

2.在数据库中执行SQL文件  (others/database/ 目录)
mysql_database_tables.sql
security-schema.sql
mysql_database_index.sql
clients.sql
users.sql
scopes.sql

3.修改 myoidc-server.properties 中的 数据库连接信息 与  oidc.issuer
4.使用TOMCAT或其他服务器启动,  访问, 初始账号/密码:  admin/password, user/password
